9.7. GPIO Interrupt Request Status Registers¶

Address 0xd8110362

GPIO Interrupt Request Status Register (GPIO[23:20])
BIT Attribute Default Description
7 RW1C 0 GPIO23 Interrupt Status When this bit is inactive (a zero), it indicates that no interrupt has been configured and detected on the associated signal. When this bit is active (an one), it indicates that an interrupt request has been configured and detected on the associated signal. Once this bit is active (an one), software must write an one to this bit to clear it inactive (a zero), otherwise this bit and the associated GPIO Interrupt Request will remain active (a one).
6 RW1C 0 GPIO22 Interrupt Status
5 RW1C 0 GPIO21 Interrupt Status
4 RW1C 0 GPIO20 Interrupt Status
